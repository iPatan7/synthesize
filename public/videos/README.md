# Videos Folder

Place your video files here:

- `doubling-the-universe.mp4` - Video for the breakthrough section background

## Video Requirements:
- Format: MP4
- Resolution: 1920x1080 or higher
- Duration: 10-30 seconds (will loop)
- File size: Keep under 50MB for web performance
- Content: Should complement the "Doubling Our Known Universe" theme

## Usage:
The video will be used as a background in the breakthrough section with:
- Blur effect (5px)
- Reduced brightness (0.7)
- Auto-play, loop, muted
- Responsive scaling
